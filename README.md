## TPS-Scanner 
### Developer Ghs Julian 
 
 This tools is for fast open port scaner.
 You can scan any websites or IP easily 

 

## Installations 

```bash
git clone https://github.com/Ghsjulian/TPScanner.git
cd TPScaner
pip3 install -r requirements.txt
python3 main.py
```
